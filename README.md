# CredVerify Global Website

Marketing website for a hybrid healthcare credentialing platform.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

Static output is generated to the `out/` directory.

## Deploy

This site can be deployed to any static hosting provider (Vercel, Netlify, etc.).
